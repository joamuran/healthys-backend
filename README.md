# Healthy's Backend

En aquest repositori tenim el backend per a una aplicació de menjars saludables, fent ús del llenguatge Dart i el framework Shelf.

El projecte serveix d'exemple per exemplificar com fer ús de Dart des del costat de backend mitjançant Shelf, i com aplicar l'arquitectura CLEAN en aquest desenvolupament.

A la carpeta [`doc`](doc/) disposeu de la documentació següent:

* [Introducció a Shelf](docs/introduccio.md)
* [Creació d'una API amb Shelf i Clean Architecture](docs/creacio_api.md)
